Accredited Educator da AWS Academy

Rodrigo Lira
Última atualização em Jan 21, 2021 1 minutos de leitura

No início de janeiro eu fui aprovado na certificação da AWS chamada AWS Certified Cloud Practitioner (ACP). Isso me garantiu o título de Accredited Educator, um credenciamento que a AWS dá aos docentes que forem aprovados no seu programa de treinamento. Dessa forma agora eu estou apto a lecionar o conteúdo da AWS Academy.

A Amazon Web Services (AWS) é um dos maiores clouds providers do mundo e possui os mais diversos serviços para computação em nuvem. Agora como um Accredited Educator, os alunos do IFPE Campus Paulista poderão realizar treinamento dessa plataforma de forma gratuita.

Meu próximo objetivo é tirar a AWS Certified Solutions Architect para em seguida alcançar a AWS Certified Machine Learning. Aproveito para deixar público o que foi utilizado no processo estudo para a certificação ACP:

  • Curso Fundamentos de nuvem da AWS

  • Bootcamp de Natal da AWS

  • Curso da Udemy com questões

  • AWS Certified cloud Practitioner training notes

  • APP AWS - Certified Cloud Practitioner. Exam CLF-C01

  • APP AWS Certified Cloud Practitioner
